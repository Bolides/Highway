PathEnvironmentParser

public struct PathEnvironmentParser : PathEnvironmentParserProtocol

Parser that extracts urls from a String-Array of paths. Usually used to parse the contents of the PATH-environment variable. Any component (PATH=comp1:comp2:…) equal to . is subsituted by $cwd. Furthermore: Any component which only contains a . is used as an input to Absolute.init:. (which standardizes the path).
